## 起動と画面構成

```sh
mkepub
```

起動すると **作家選択画面** が表示されます。画面は以下の流れで遷移します：

```
作家選択 → 作品選択 → 変換中
              ↓
          メール送信
```

---

## 作家選択画面

青空文庫に収録されている作家の一覧が表示されます。

### 操作

| キー | 動作 |
|------|------|
| 文字入力 | 作家名を絞り込み（姓名・読み・ローマ字対応） |
| `↑` `↓` | カーソル移動 |
| `←` `→` | ページ移動 |
| `Enter` | 選択した作家の作品一覧へ |
| `m` | メール送信画面へ（設定済みの場合のみ） |
| `q` / `Esc` | 終了 |

### 絞り込み

入力欄に文字を入力すると、以下の項目で部分一致検索します：

- 姓名（例：`夏目漱石`）
- 読み（例：`なつめ`）
- ローマ字（例：`natsume`）

---

## 作品選択画面

選択した作家の作品一覧が表示されます。複数作品をまとめて変換できます。

### 操作

| キー | 動作 |
|------|------|
| `↑` `↓` | カーソル移動 |
| `←` `→` | ページ移動 |
| `Space` | 作品を選択／解除 |
| `Enter` | 選択した作品を変換開始 |
| `Esc` | 作家選択に戻る |

### 表示項目

```
[x] 坊っちゃん            1906  913
 ^  ^                     ^     ^
 |  タイトル（最大36文字） |     NDC分類
 選択状態                  発行年
```

すでにEPUBが生成済みの作品には `✓` マークが表示されます。

---

## 変換中画面

選択した作品を順番にダウンロード・変換します。

```
変換中

  …  坊っちゃん        ← 処理中
  ✓  /path/to/file.epub  ← 完了
  ✗  吾輩は猫である     ← エラー
     エラーメッセージ
```

すべての変換が完了したら `Enter` で作家選択画面に戻ります。

---

## メール送信画面

`output_dir` に保存されているEPUBファイルを選択してメールで送信します。
Kindleへの送信（Send to Kindle）に利用できます。

### 前提条件

設定ファイルの `[mail]` セクションに `from` と `to` が設定されていること。

### 操作

| キー | 動作 |
|------|------|
| `↑` `↓` | カーソル移動 |
| `←` `→` | ページ移動 |
| `Space` | ファイルを選択／解除 |
| `Enter` | 選択したファイルを送信 |
| `Esc` | 作家選択に戻る |

作家選択画面で `m` キーを押すと遷移します（`from` / `to` が未設定の場合は無効）。

---

## 設定ファイル

`~/.config/mkepub/config.toml`

```toml
# EPUBの出力先ディレクトリ
output_dir = "~/.local/share/epub"

# 青空文庫CSVのパス
csv_path   = "~/.local/share/mkepub/list_person_all_extended_utf8.csv"

[mail]
smtp_host = "smtp.gmail.com"
smtp_port = 587
from      = "your@gmail.com"   # 送信元アドレス
password  = "xxxx xxxx xxxx"   # アプリパスワード
to        = "your@kindle.com"  # 送信先アドレス（例：Kindle）
```

### Gmailを使う場合

Googleアカウントの「アプリパスワード」を生成して `password` に設定してください（通常のパスワードは使用不可）。

### パスの記法

`output_dir` と `csv_path` は `~/` による相対記法に対応しています。

---

## 出力ファイル

変換後のEPUBは以下の命名規則で保存されます：

```
{作者名}_{タイトル}.epub
```

例：
```
夏目漱石_坊っちゃん.epub
夏目漱石_吾輩は猫である.epub
```

ファイル名に使用できない文字（`/ \ : * ? " < > |`）は全角文字に置換されます。
